# **geoinfobage**

**Project Status: In development** :warning:

# **Description**
geoinfobage is a WEB APP application using the R language, HTML, CSS and Javascript through the Shiny and Leaflet packages allowing users to easily access spatial data sets in the city of Bag√©/RS. The application includes a wide variety of geospatial data as simple features. They were obtained from several sources for the acquisition of the original files in shapefile, later converted into GeoJSON, other geospatial data were generated by the GeoInformation sector.
